Inputs: Radios

Radio buttons allow the user to select one option from a set.

Radio button 1

Radio button 2

Radio button 3

This is a help text.

<template>
  <SInputRadios
    name="input"
    label="Label"
    note="Note text"
    help="This is a help text."
    :nullable="false"
    :options="options"
    v-model="form.input"
  />
</template>

<script>
import SInputRadios from '@@/lib/components/SInputRadios'

export default {
  components: {
    SInputRadios
  },

  data () {
    return {
      form: {
        input: 1
      },

      options: [
        { label: 'Radio button 1', value: 1 },
        { label: 'Radio button 2', value: 2 },
        { label: 'Radio button 3', value: 3 }
      ]
    }
  }
}
</script>

API

Props

NAME

TYPE

REQUIRED

DEFAULT

DESCRIPTION

NAME

name

TYPE

String

REQUIRED

No

DEFAULT

null

DESCRIPTION

The name for the field. It will be set to `for` attribute at label, and `id` attribute at input so that when user clicks the label, the input will get focus.

NAME

label

TYPE

String

REQUIRED

No

DEFAULT

null

DESCRIPTION

The label for the input.

NAME

note

TYPE

String

REQUIRED

No

DEFAULT

null

DESCRIPTION

The small note text that will be displayed next to label. The note should be short for example `Optional` or `Max length 160`.

NAME

help

TYPE

String

REQUIRED

No

DEFAULT

null

DESCRIPTION

The help text that will be displayed under the input. Useful to add a little detailed information about the input.

NAME

nullable

TYPE

Boolean

REQUIRED

No

DEFAULT

true

DESCRIPTION

Whether users can check checked radio button to uncheck it or not. It means, users may have no radio button selected for the form.

NAME

options

TYPE

Array

REQUIRED

Yes

DEFAULT

—

DESCRIPTION

The available options for the radio button group. The array should contain object with properties of `label` and `value`. `label` is used as a text for the radio button, and `value` is the value corresponding to the radio button.

NAME

value

TYPE

String | Number | Boolean

REQUIRED

No

DEFAULT

—

DESCRIPTION

The currently selcted value of the radio button group.

Events

NAME

DESCRIPTION

NAME

change

DESCRIPTION

Fires when a user clicks a radio button.
